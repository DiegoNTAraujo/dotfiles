# 🧰 Dotfiles

My personal **dotfiles** for an **Arch Linux + Hyprland** setup.
> ⚠️ These configurations are tailored to my hardware and preferences. Use them as reference or inspiration.

## 🖥️ System

**Personal Laptop**

- **OS:** Arch Linux
- **WM:** Hyprland
- **RAM:** 24GB
- **CPU:** i5-13450HX
- **GPU:** RTX 3050

## 🔧 Dependencies

**Some of the main packages used:**

- hyrpland
- hyprlock
- hypridle
- hyprpaper
- hyprshot
- waybar
- rofi
- alacritty
- brightnessctl
- gammastep
- ttf-jetbrains-mono-nerd
- otf-font-awesome

**Arch**:
```
sudo pacman -S hyprland hyprlock hypridle hyprpaper hyprshot waybar rofi alacritty brightnessctl gammastep ttf-jetbrains-mono-nerd otf-font-awesome
```
